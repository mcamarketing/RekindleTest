- **Headings:** System font stack (SF Pro Display / Inter), bold, -0.03em letter spacing

- 4px base unit (xs)
- 8px (sm)
- 16px (md)
- 24px (lg)
- 32px (xl)

- **Cards:** White bg, 1px border #e3e8ee, 8-12px rounded, 0_2px_4px_rgba(0,0,0,0.04) shadow
- **Buttons:**

  - Destructive: #ef4444 bg, white text
- **Inputs:** White bg, #e3e8ee border, focus:ring-2 ring-orange-500/20
- **Badges:** Small pills, subtle colors, never garish
- **Animations:** 200-300ms ease, subtle scale/fade only
